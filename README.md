
 Project Overview

This project focuses on analyzing bank loan data to uncover meaningful insights related to loan performance, customer behavior, and financial risk. Using SQL and Power BI, the project transforms raw loan data into interactive dashboards that support data-driven decision-making for banking and financial institutions.

The analysis highlights key metrics such as loan applications, funded amounts, repayment trends, interest rates, and borrower risk classification.

 Objectives

Analyze overall loan performance and trends

Classify loans into Good and Bad categories

Understand borrower demographics and financial behavior

Track Month-to-Date (MTD) and Month-over-Month (MoM) performance

Build interactive dashboards for executive-level insights

 Dashboards & Insights

 Key Insights Derived

Over 86% of loans fall under the Good Loan category

Debt consolidation is the most common loan purpose

Majority of loans are issued for 36 months

Higher interest rates are strongly associated with charged-off loans

Borrowers with lower DTI show better repayment performance
